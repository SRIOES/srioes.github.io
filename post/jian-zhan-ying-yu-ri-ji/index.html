建站英语日记

Have you tried setting up a website? I have.

On 2019.7.13 , we set up a group named SRIES (The second Research Institute of experimental school) on QQ. We talk about many interesting things there, included studying, science, physics and chemistry and so on (this is not the main theme of this article, I’ll write about it in another article in future).Recently, After seeing a website of my friend, I decided to set up a website of mine or SRIES’s.
I used a website named github to use its pages service and used a software named “gridea” to build my website and write articles (very easy). 	Firstly, I set up a website of mine easily, its website is https://exwatermelon.github.io/ and I wrote an article named “谈一点对新裤子乐队的认识” on April Third.
After that, I used gridea to set up a website of SRIES on April 18th. Its website is https://sries.lwzheng.icu .I not only copied the article “谈一点对新裤子乐队的认识” to this website, but also copied the small story that written by Bao Zheng on https://sries.lwzheng.icu. Although I just used a simple software to set up a website easily, I believe that I can set up a website of ours by codes and programming in the future, it’s also a dream of mine.
DON’T FORGET TO VISIT OUR WEBSITE AT https://sries.lwzheng.icu !!!!!!!!!!!!
